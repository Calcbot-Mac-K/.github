## 🔍 Overview

<a href="#calcbot">Calcbot</a> is Tapbots' thoughtfully designed calculator for Mac — bringing genuine design attention and practical capability improvements to daily calculation that most Mac users accept from the default Calculator without considering that it could be significantly better.

The <a href="#calcbot">calculation tape</a> is Calcbot's most important feature over the default Mac Calculator — every calculation entered during a session appears as a scrollable tape showing each expression and its result, remaining visible and accessible throughout the session. Any previous result taps to reuse immediately, any previous step is reviewable for verification, and the complete session history is retained until cleared. The default macOS Calculator shows only the current number — one result at a time with no history — making multi-step calculations where intermediate results matter require manual note-taking. Calcbot replaces that limitation with a persistent visual record of the complete calculation session. The <a href="#calcbot">live expression preview</a> shows the calculated result of the current expression as it is being entered, before pressing equals — typing 15 + 27 × 3 shows the correct result updating in real time as each character is added, providing immediate feedback that the expression is being entered correctly before committing.

The <a href="#calcbot">unit and currency conversion</a> integrates conversion directly into the calculator workflow — converting between measurement units and currencies without switching to a separate application or web browser. Length, weight, temperature, volume, speed, area, and other measurement categories convert through the same interface used for standard calculations. The <a href="#calcbot">correct commercial percentage handling</a> calculates percentages intuitively — 200 + 15% adds 30 to 200 for a result of 230, matching the expected behavior for sales tax, tips, and discounts rather than the mathematically literal but practically wrong result that macOS's default Calculator produces.

<a href="#calcbot">Custom themes and appearance</a> provides multiple visual styles for the calculator — light, dark, and color options that make a tool opened dozens of times daily pleasant to use rather than a utility to tolerate. <a href="#calcbot">iCloud sync across Apple devices</a> keeps calculation history and settings consistent across Mac, iPhone, and iPad — a calculation started on iPhone accessible on Mac, and the complete session history available on any device.

<a href="#calcbot">Keyboard shortcut operation</a> makes every calculator function accessible without mouse interaction — operators, functions, and navigation all reachable from the keyboard for calculation-intensive workflows where mouse use disrupts flow. <a href="#calcbot">Apple Watch companion</a> shows recent calculations and the current result on the wrist for quick reference without returning to the Mac or iPhone.
